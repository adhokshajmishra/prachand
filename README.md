# Prachand

#### What is it?
Bare minimal C2 and payload skeleton framework

#### Why is it a thing?
Because I wanted to combine my scattered code in one place, in one unified framework. This is the skeleton part of that project.

#### What can I do with it?
- Plug your own payload, and use a functional C2
- Automate red team
- Learn Linux MalDev

## How to contribute
Contributions are always welcome.

#### Code
Open a PR with your code contribution.

#### Issues
If you want to report an issue, include the following:

- Linux distribution name and version
- Compiler and version
- Boost and Libpqxx versions
- Steps to reproduce

#### Documentation
Open a PR with your documentation contribution
